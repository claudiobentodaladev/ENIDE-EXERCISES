# ENIDE Exercises Repository

This repository contains various practical **ENIDE** exercises aimed at reinforcing the main concepts of the language.

## About ENIDE

ENIDE programming language — a multilang language that allows programming over 100 natural language, removing the language barrier in learning to program.
It features a syntax similar to the most popular programming languages, aiding in further studies.

## Repository Purpose

- Practice Enide fundamentals
- Improve programming logic
